# Loan Request App

A simple React application that simulates a loan request process through an interactive form.

Live Demo
🔗 https://loan-quest-app.vercel.app/



## Features

* User information form (Name, Phone Number, Age)
* Employee status selection
* Salary range selection
* Form validation
* Custom popup notifications
* Conditional rendering
* State management using React Hooks
* React Router integration

## Technologies Used

* React.js
* JavaScript (ES6+)
* React Hooks (`useState`)
* React Router
* CSS3

## Getting Started

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project folder:

```bash
cd loan-request-app
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm start
```
